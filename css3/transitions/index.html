CSS3 Transitions

Using transition for :focus and :hover effects. This demo uses SVG images, which at this stage are not supported in all browsers.


a { ...
    height: 75px;
    padding-top: 75px;
    -o-transition: all 0.3s ease-in-out;
    ...
}

a:focus, a:hover {
    height: 150px;
    padding-top: 0px;
}

a img {
    width: 75px;
    height: 75px;
    -o-transition: all 0.3s ease-in-out;
    ...
}

a:focus img, a:hover img {
    width: 150px;
    height: 150px;
}
