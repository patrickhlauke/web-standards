Video letterboxing

If video file and video element's dimensions don't have same aspect ratio, browsers letterbox, rather than stretch.

Video file is 426x240px, approximately 16:9 ratio. Video element is set to 450x335px, approximately 4:3 ratio. In this case, the video is scaled to fill the width, and then letterboxed top and bottom. The letterboxed areas use whatever background applies to video according to CSS (transparent by default).

video { display: block; border: 1px #999 solid; background: #fcc; }
